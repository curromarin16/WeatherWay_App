# WeatherWay 🌦️🗺️

Aplicación móvil desarrollada en **Flutter** para descubrir lugares cercanos en función del clima actual.

Este proyecto forma parte de un **Trabajo de Fin de Grado (TFG)** y combina geolocalización, mapas, datos meteorológicos y persistencia en la nube para ofrecer recomendaciones útiles al usuario.

---

##  Funcionalidades principales

- Registro e inicio de sesión con **Firebase Auth** (email/contraseña y Google).
- Obtención de ubicación del usuario con **Geolocator**.
- Visualización de mapa interactivo con **Google Maps**.
- Recomendación de lugares según clima y categorías.
- Consulta de clima actual mediante **OpenWeatherMap**.
- Guardado de favoritos en **Cloud Firestore**.
- Pantallas de perfil, ajustes y gestión de favoritos.

---

## Tecnologías utilizadas

- **Flutter / Dart**
- **Firebase Core**
- **Firebase Auth**
- **Cloud Firestore**
- **Google Sign-In**
- **Google Maps Flutter**
- **Google Maps Web Services**
- **Geolocator**
- **HTTP**
- **Connectivity Plus**
